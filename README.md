\# 🤖 5-DOF Robotic Manipulator for Chocolate Placement



\## 📌 Project Overview



This project focuses on the \*\*design and development of a 5-DOF robotic manipulator\*\* for automated chocolate placement. The robot is designed to pick chocolates from a designated area using sensors and place them accurately into a \*\*10×10 grid tray\*\* to form a structured arrangement.



The system emphasizes \*\*kinematic modeling, mechanical design, and end-effector development\*\* for precise pick-and-place operations.



\---



\## 🎯 Objectives



\* Design a \*\*5-DOF robotic manipulator\*\*

\* Perform accurate \*\*pick-and-place operations\*\*

\* Develop \*\*forward and inverse kinematics\*\*

\* Design a \*\*gripper-based end-effector\*\*

\* Implement \*\*sensor-based object detection for picking\*\*

\## 🧮 Kinematic Modeling



The robot uses:



\* \*\*Forward Kinematics\*\* to determine end-effector position from joint angles

\* \*\*Inverse Kinematics\*\* to compute joint angles for a desired position



\---



\## 🏗️ Design Specifications



\* Tray Size: \*\*20 cm × 20 cm\*\*

\* Grid: \*\*10 × 10 (100 positions)\*\*

\* Chocolate Shape: Ellipsoid



&#x20; \* Major Axis: \*\*10 mm\*\*

&#x20; \* Minor Axis: \*\*6 mm\*\*



\---



\## 🔄 Working Principle



1\. Sensor detects chocolate in pick-up area

2\. Robot moves to pick position

3\. Gripper grasps the chocolate

4\. Robot moves to target grid position

5\. Chocolate is placed accurately in tray
